# Google-Books-Search
#### Author - Scott Nelson

This is an app for searching Google Books. It allows the user to save books that they are interested in for a later time and lets them delete books from their list if they are no longer interested.

## Technologies Used
#### Backend
* Node
* Node Fetch
* Dotenv
* Express
* MongoDB
* Mongoose
* Morgan (logging)
* Concurrently

#### Frontend
* React
* Axios
* Bootstrap
* Fontawesome
* HTML5
* CSS3
* Javascript
